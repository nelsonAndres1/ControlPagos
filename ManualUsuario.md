# Manual de Usuario - ControlPagos

Bienvenido a tu guía de usuario para **ControlPagos**. Este manual explica cada opción del menú, lo que hace y cómo usarla.

> Nota: este manual ahora también está disponible desde la aplicación en la opción **Manual de Usuario** del menú.

---

## 1. Acceso al sistema

### 1.1. ¿Dónde entro?

- Abre el navegador e ingresa la URL del sistema.
- Verás la pantalla de login.
- Ingresa tu usuario y contraseña.
- Si no tienes credenciales, pídelas al administrador del sistema.

### 1.2. ¿Qué ocurre después de iniciar sesión?

- Si el ingreso es correcto, llegarás al menú principal.
- El menú muestra sólo las opciones habilitadas para tu usuario.
- Si no puedes ver una opción, probablemente no tengas permiso para esa función.

---

## 2. Menú principal y opciones del sistema

A continuación se describen las opciones visibles para el usuario final, con su propósito y cómo usarlas.

### 2.1. Página Principal

- Esta es la pantalla inicial después del login.
- Muestra un semáforo de pagos pendientes.
- Permite filtrar por prioridad, buscar pagos y ver detalles rápidos.
- Ideal para identificar qué pagos requieren atención inmediata.

### 2.2. Inicie el Pago

- Sirve para iniciar o crear un nuevo pago.
- Usa esta opción cuando quieras cargar un pago nuevo en el sistema.
- Puede incluir datos básicos del pago y su proceso de autorización.

### 2.3. Editar sus pagos

- Permite revisar y modificar pagos ya registrados.
- Usa esta opción si necesitas corregir datos de un pago existente.
- Solo está disponible si tu usuario tiene permiso de edición.

### 2.4. Editar soportes de pago

- Sirve para actualizar los soportes (documentos) asociados a un pago.
- Útil cuando los comprobantes, facturas o soportes cambian o se deben anexar nuevos.

### 2.5. Reimprimir pago

- Permite reimprimir un comprobante o documento de pago.
- Útil cuando se necesita generar otra copia del registro.

### 2.6. Consultar modificaciones

- Muestra el historial de cambios realizados sobre un pago.
- Úsalo para ver qué se modificó, cuándo y quién lo hizo.

### 2.7. Consultar estado del Pago

- Busca el estado actual de un pago.
- Ideal cuando quieres saber si un pago está aprobado, rechazado, en proceso o pendiente.

### 2.8. Consultar por Nit

- Busca pagos asociados a un NIT específico.
- Útil para consultas por proveedor o entidad.

### 2.9. Modificar estado del Pago

- Permite cambiar el estado de un pago.
- Útil para procesos de autorización, reversión o avance de etapas.
- Requiere permiso especial.

### 2.10. Tipos de Pago

- Configura los tipos de pago que usa el sistema.
- Ejemplo: efectivo, transferencia, cheque, etc.
- Solo disponible para usuarios de administración.

### 2.11. Tipos de Soportes

- Configura las clases de soporte que se pueden usar en pagos.
- Ejemplo: factura, comprobante, soporte bancario, etc.

### 2.12. Editar Soportes

- Permite cambiar las propiedades del catálogo de soportes.
- Útil para mantener actualizada la lista de documentos válidos.

### 2.13. Vincular Pagos y Soportes

- Relaciona los pagos con los soportes que los respaldan.
- Se usa cuando tienes un pago y necesitas asociar o validar sus comprobantes.

### 2.14. Roles de Menú

- Permite asignar o revisar permisos de menú.
- Solo para administradores con perfil de menú.

### 2.15. Agregar Causadores

- Permite registrar causadores o usuarios que generan pagos.
- Usado para mantener la lista de responsables de origen.

### 2.16. Revisores y Autorizadores

- Gestiona los usuarios encargados de revisar y autorizar pagos.
- Útil para definir quién puede aprobar o rechazar solicitudes.

### 2.17. Crear Opciones

- Crea opciones de proceso o etapas internas.
- Suele usarse para parametrizar pasos de flujo.

### 2.18. Crear Arbol

- Construye estructuras de proceso en forma de árbol.
- Se utiliza para definir cómo se agrupan o relacionan las etapas de pago.

### 2.19. Árbol y Pagos

- Relaciona el árbol de proceso con los pagos.
- Permite ver o administrar el flujo de pagos dentro de la estructura creada.

### 2.20. Permisos

- Muestra las opciones de permiso legacy / históricas.
- Sirve para revisar o ajustar permisos antiguos.

### 2.21. Notificaciones

- Configura notificaciones automáticas para pagos y pasos.
- Puedes asociar un correo con un pago y uno o varios pasos.
- El sistema envía alertas cuando ocurren eventos relevantes.

### 2.22. Reportes

- Accede a reportes predefinidos del sistema.
- Ideal para extraer información estándar de pagos, estados y autorizaciones.

### 2.23. Reportes Dinámicos

- Permite crear consultas personalizadas.
- Útil para obtener información específica según filtros y criterios.

### 2.24. Tablero

- Pantalla de seguimiento visual.
- Muestra indicadores y gráficos del estado general de los pagos.
- Ideal para supervisores y gerentes.

### 2.25. Pasos Generales

- Administra pasos generales del proceso de pago.
- Se usa para ver y ajustar el flujo de trabajo.

### 2.26. Administrador Soportes

- Panel de administración para los soportes de pago.
- Permite gestionar catálogos o módulos de soporte.

### 2.27. Chat

- Chat interno para comunicarse con otros usuarios.
- Incluye:
  - Crear nueva conversación directa o de grupo.
  - Agregar participantes.
  - Ver conversaciones existentes.
  - Enviar mensajes en tiempo real.

### 2.28. Salir

- Cierra tu sesión de usuario.
- Siempre usa esta opción al terminar tu trabajo.

## 3. Cómo usar cada opción

### 3.1. Página Principal

- Al entrar, verás un resumen de pagos pendientes.
- Filtra por prioridad usando los botones del semáforo.
- Busca por código, número, estado o clase.
- Selecciona la prioridad para ver solo los pagos de ese grupo.

### 3.2. Iniciar Pago

- Haz clic en **Inicie el Pago**.
- Carga los datos del pago.
- Envía o guarda el registro según el flujo.

### 3.3. Editar pagos

- Haz clic en **Editar sus pagos**.
- Busca el pago que quieras actualizar.
- Cambia los campos necesarios y guarda.

### 3.4. Editar soportes de pago

- Selecciona **Editar soportes de pago**.
- Cambia o agrega documentos que respalden el pago.
- Guarda para que la relación soporte-pago quede actualizada.

### 3.5. Reimprimir pago

- Selecciona **Reimprimir pago**.
- Busca el pago.
- Genera la impresión o descarga del comprobante.

### 3.6. Consultar modificaciones

- Selecciona **Consultar modificaciones**.
- Revisa el historial de cambios.
- Útil para auditoría y seguimiento.

### 3.7. Consultar estado del Pago y Consultar por Nit

- Usa **Consultar estado del Pago** para ver el estado general.
- Usa **Consultar por Nit** para buscar todos los pagos asociados a un NIT.

### 3.8. Modificar estado del Pago

- Selecciona la opción y busca el pago.
- Cambia el estado según lo que corresponda: aprobado, rechazado, pendiente, etc.

### 3.9. Notificaciones

- Selecciona **Notificaciones**.
- Busca el usuario por documento o nombre.
- Ingresa el correo del destinatario.
- Elige el pago o pagos asociados.
- Selecciona los pasos en los cuales debe llegar la notificación.
- Guarda la configuración.

### 3.10. Chat

- Ve a **Chat**.
- Crea una conversación: elige título, tipo y participantes.
- Selecciona cualquier conversación para entrar al chat.
- Envía mensajes con el campo inferior.
- Si tienes mensajes nuevos, el icono mostrará el número de unread.

---

## 4. Recomendaciones para el usuario

- Navega siempre desde el menú lateral para no perderte.
- Usa las búsquedas cuando haya muchos registros.
- Si un botón no aparece, pregunta al administrador por permisos.
- No modifiques datos si no estás seguro de su impacto en el pago.
- Cierra sesión al finalizar.

---

## 5. Casos de uso comunes

### 5.1. Quiero registrar un pago nuevo

1. Ingresa al sistema.
2. Haz clic en **Inicie el Pago**.
3. Completa los datos del pago.
4. Guarda o envía según el formulario.

### 5.2. Quiero verificar un pago asociado a un NIT

1. Ingresa al sistema.
2. Haz clic en **Consultar por Nit**.
3. Escribe el NIT y ejecuta la búsqueda.

### 5.3. Quiero hablar con mi equipo sobre un pago

1. Haz clic en **Chat**.
2. Crea una conversación directa o grupal.
3. Agrega participantes.
4. Envía tu mensaje.

---

## 6. Permisos y visibilidad

- Las opciones del menú aparecen según los permisos de tu usuario.
- Si no ves una opción, puede que tu rol no tenga acceso.
- Consulta con el administrador para obtener permisos adicionales.

---

## 7. Consejos técnicos básicos

- Si la carga es lenta, actualiza la página.
- Si algo no funciona, revisa si hay un mensaje de error en pantalla.
- Siempre usa el botón **Salir** cuando termines.
